# Intervals

Ear training for recognizing musical intervals.

## Development

```bash
npm install
npm run dev
```

The Vite dev server runs on port `5173`.

## DevPod

```bash
devpod up .
devpod ssh intervals
```

The canonical container name is `intervals-devcontainer`.

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run test
```
